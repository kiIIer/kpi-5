# Лабораторна робота №5

## Тема

Створення віртуального символьного пристрою у варіанті модуля ядра Linux
і прототипів системних викликів

## Виконання

Для виконання завдання я створив модуль ядра, який реалізує операцію read для передачі випадкового набору біт у простір користувача. Реалізація включає в себе свторення символьного пристрою, що генерує випадкові біти при відкритті та повертає їх при читанні.

## Висноок

У цій лабораторній роботі я ознайомився з процесом розробки, компіляції, тестування та вивантаження модулів ядра Linux. Особливу увагу було приділено роботі з символьними пристроями та взаємодії між простором ядра та простором користувача. Розуміння цих процесів є ключовим для розробки ефективних та надійних системних рішень на рівні ядра операційної системи.
